## 📊 Project Overview

This project predicts the pathogenicity of genetic variants using machine learning.
It integrates biological features such as:
- Allele Frequency
- CADD Score
- REVEL Score
- SIFT Score
- PolyPhen Score
- PPI Score

The model is deployed using Streamlit for real-time predictions.
